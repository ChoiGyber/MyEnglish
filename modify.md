### `e4ffe48` — 오답 기록·복습·공유·모달 동작 구현
이번 단계에서 가장 큰 변경.

**상태 모델 확장**
- `S.wrong` 맵 추가 — `"chapterId:qIdx"` → 누적 오답 횟수
- 정답 시 `-1`, 오답 시 `+1` (0 미만은 0으로 클램프, 0이면 키 삭제)

**퀴즈 엔진 리팩터**
- `quizState.items` 도입 — `[{chId, qIdx, q}]` 형태로 출제 순서 관리
- `startQuiz()` 가 단원의 문항을 **오답 횟수 내림차순으로 재정렬**
  → 자주 틀린 문제가 앞쪽에 배치
- `renderQ()` 에서 `wrongCount >= 1` 인 문항은 카드에 빨간 보더 + `⚠️ 복습 문제 — 지난번 N번 틀렸어!` 배지 표시
- 정답 시 "복습 완료! 👏" 부가 메시지

**복습 모드 신설**
- `startReviewQuiz()` — 모든 단원에서 자주 틀린 문항을 모아 출제
- `quizState.isReview` 플래그로 분기
- 결과 화면에서 단원 진도·별점에 영향을 주지 않음
- XP/똔 **1.5배 보너스**

**복습 배너**
- `updateReviewBanner()` — 오답 카운트가 1개 이상일 때만 표시
- 클릭 시 `startReviewQuiz()` 진입

**카카오톡 공유**
- `buildShareMessage()` — 레벨, 똔, 별, 클리어 단원 수를 포맷팅
- `shareToKakao()` — 3단계 폴백:
  1. `navigator.share` (모바일에서 카톡 선택 가능)
  2. `navigator.clipboard.writeText` (PC에서 복사 후 붙여넣기)
  3. `prompt()` 로 마지막 폴백
- 결과 화면 공유는 방금 푼 단원/점수도 메시지에 포함

**환영 모달 동작**
- `openWelcome()` / `closeWelcome()`
- `❓` 버튼으로 언제든 재오픈 가능
- 첫 방문 시 자동 노출 (`gqVisited` 플래그로 제어)

같은 PC에서 자녀 여러 명이 각자 진도를 쌓을 수 있도록.

### `3122af5` — 프로필 모달과 HUD 칩 마크업
- 프로필 선택 모달(`#profileMask`) — 프로필 카드 그리드 + 새 프로필 폼(숨김 상태)
- HUD에 현재 프로필 칩 추가 (이모지 + 이름)

### `99db4c7` — 프로필 시스템 구현
**저장소 구조 변경**

| 키 | 의미 |
|---|---|
| `gqProfiles` | `[{id,name,emoji}]` 프로필 목록 |
| `gqCurrentProfile` | 현재 활성 프로필 id |
| `gqState_<id>` | 해당 프로필의 학습 상태 JSON |
| `gqVisited_<id>` | 해당 프로필의 환영 모달 1회 표시 플래그 |

**핵심 함수**
- `loadProfiles()` / `saveProfiles()` — 프로필 목록 입출력
- `currentProfile()` / `stateKey()` / `visitedKey()` — 현재 프로필 기반 키 생성
- `setCurrentProfile(id)` — 활성 프로필 전환 + 상태 재로드 + UI 갱신
- `deleteProfile(id)` — 프로필과 모든 관련 키 삭제
- `migrateLegacyIfNeeded()` — 구버전 단일 저장소(`grammarQuestStateV1`) 가 있으면 "지유 🦄" 프로필로 자동 이관

**UX**
- 프로필이 없으면 첫 실행 시 모달이 강제 노출 (× 버튼 숨김)
- 프로필 카드 미리보기 — `Lv.X · YYY 똔`
- 카드 호버 시 우상단 × 로 삭제 (확인 후 진행)
- 새 프로필: 이름(≤8자) + 이모지 10종 중 선택
- HUD 칩 클릭으로 언제든 프로필 전환
- 환영 모달 제목과 카톡 공유 메시지에 프로필 이름 자동 반영

| 환영 모달 | `+5 JY` | `+5 똔` |
| 복습 배너 | `+ 보너스 JY!` | `+ 보너스 똔!` |
| 배지 | `부자 지유 — 지유원 500 JY 모으기` | `부자 — 똔 500개 모으기` |
| 공유 메시지 | `💰 지유원: ${gold} JY` | `💰 똔: ${gold}개` |
| 프로필 카드 메타 | `Lv.X · 0 JY` | `Lv.X · 0 똔` |

추가로 한글 단위 가독성을 위해 `.unit` 폰트 사이즈를 10px → 13px로 키움.

## 최종 기능

- 12단원 영문법 RPG 학습 (별·콤보·레벨·배지)
- 정답 시 **똔** 적립 — 현금 교환 안내
- 자주 틀린 문제 자동 추적 + 다음 학습 시 앞으로 재배치 + 시각 강조
- 전 단원에서 자주 틀린 문제만 모아 푸는 **복습 모드** (보너스 +50%)
- 결과/홈에서 **카톡으로 아빠한테 자랑하기**
- 기기당 자녀별 **다중 프로필** (진도·똔·오답·뱃지가 각자 독립)
- 첫 방문 환영 모달, `?` 버튼으로 재오픈 가능

## 알려진 한계 / 향후

- **로컬 저장 한정** — 다른 기기에서 접속하면 진도가 따라오지 않음. 클라우드 동기화는 Firebase/Supabase 무료 플랜으로 추가 가능.
- **개발자도구로 점수 조작 가능** — 신뢰 기반 운영. 절대 못 속이게 하려면 서버 검증 필요.
- 카톡 공유는 모바일에서 네이티브 공유 시트가 뜨고, PC에서는 클립보드 복사로 폴백.
